Tilt

Tilt is a JavaScript and CSS library for building Metro navigation grids, tiles and other things that tilt. It mimics the behavior of Metro tiles which when pushed tilt based on the interaction location.

Features

  • Responsive

    It's responsive and therefor works on a wide range of modern mobile, tablet and desktop browsers.

  • Physics

    It mimics the interaction effects of Windows Metro. Excellent for mobile touch feedback.

  • Metro Grids

    It comes with a smart grid engine for generating metro style grids on any resolution.

  • Customizable

    The package contains the original source files so you can tweak it to your own liking.
