Tilt

Hi!

Welcome to tilt.

Tilt is a CSS and JavaScript library for building Metro like tiles and other things that tilt. It mimics the behaviour of 3D Metro UI tiles which when pushed tilt based the interaction location.

It's responsive, and works on a wide range of modern mobile, tablet and desktop browsers.

Below you can view some demos created with tilt, they should give you an impression on how you could apply tilt in your project.

This page also features the tilt docs. They contain a complete overview of what tilt brings to the table.

A grid containing tiles which feature 3D interaction patterns similar to Metro UI.

Node

You can tilt any html element by adding the .tilt classname to it's class attribute.

It's best to wrap the element being tiltd in another element. See the example below. The anchor is being tiltd and it's wrapped in a div element.

tilt classname on an anchor
<div>
    <a class="tilt">Foo</a>
</div>

Tile

Create Metro style tiles by adding the .tilt-tile classname to an html element. If you want 3D interaction patterns, leave the .tilt classname on the element.

tilt tile example
<a class="tilt tilt-tile">Foo</a>
  • Foo

Caption

Within the tile you can align the caption. By using the following classnames.

  • .tilt-caption-tl to position the text in the top left corner, this is the default.
  • .tilt-caption-tr positions the text to the top right.
  • .tilt-caption-tc top center.
  • .tilt-caption-br bottom right.
  • .tilt-caption-bl bottom left.
  • .tilt-caption-bc and the last one, bottom center.

If there's more text than space available the text will be cutoff using ellipsis...

If you want to hide the caption, add the .tilt-caption-hidden classname.

Note that to position the caption you need to wrap the caption in a <span> element as shown in the html snippet below.

tilt caption example
<a class="tilt tilt-tile tilt-caption-br">
    <span>Foo</span>
</a>

Icons

The icons are setup using Font Awesome, a free to use icon font, it's possible to use your own icons though.

You can have the icon separate from the text or linked to the text, in the later case it's not possible to use any of the positioning options.

tilt icon example
<!-- icon separate from text -->
<a class="tilt tilt-tile tilt-caption-br">
    <i class="fa fa-coffee"></i>
    <span>Separate</span>
</a>

<!-- icon linked to text -->
<a class="tilt tilt-tile tilt-caption-br">
    <span><i class="fa fa-coffee"></i> Linked</span>
</a>

Position

You can set the icon to the center of the tile by adding the .tilt-icon-center classname.

On top of this you can move it to all corners of the tile by adding the following classnames.

  • .tilt-icon-tl for position top left.
  • .tilt-icon-tr for top right.
  • .tilt-icon-bl bottom left.
  • .tilt-icon-br and bottom right.

If you've moved your caption into any of these positions tilt will automatically make space for the icon.

tilt icon position example
<!-- centered icon -->
<a class="tilt tilt-tile tilt-icon-center">
    <i class="fa fa-coffee"></i>
    <span>Coffee</span>
</a>

<!-- icon bottom right corner -->
<a class="tilt tilt-tile tilt-icon-br tilt-caption-br">
    <i class="fa fa-coffee"></i>
    <span>Coffee</span>
</a>

<!-- icon in top right corner -->
<a class="tilt tilt-tile tilt-icon-tr tilt-caption-br">
    <i class="fa fa-coffee"></i>
    <span>Coffee</span>
</a>

Image

The .tilt-img classname can be used to set images as the background of a tilt element.

If the caption <span> is placed after <img> it is automatically given a black translucent color. If the caption precedes the element the background is left transparent.

tilt tile image example
<!-- black background, <img/> first -->
<a class="tilt tilt-tile tilt-img">

    <img src="foo.jpg"/>

    <span>Foo</span>

</a>

<a class="tilt tilt-tile tilt-img">

Corners

If you prefer the corners of the tile to be a little less sharp, add one of the following pillow classes.

  • .tilt-pillow-s for tiny corners.
  • .tilt-pillow-m or .tilt-pillow for medium corners.
  • .tilt-pillow-l to make the corners really stand out.

Tiles

Tilt enables you to define tile types, based on the type of tile defined, the grid will be rearranged.

There's four different square tiles, two landscape and two portrait tiles.

  • s, m, l and xl render square tiles.
  • _s and _l render landscape tiles.
  • |s and |l render portrait tiles.

<ul class="tilt-grid">
    <li data-tile-type="m">

        Bar

        Baz

        Foo
    </li>

        Bar
    </li>
    <li data-tile-type="|s">
        Baz

</ul>

Hints

Now this is where it gets a little tricky. When you resize the screen, tilt will try to rearrange the tiles so they still fit nicely on the grid.

Unfortunately tilt is not SkyNet so sometimes it needs a little nudge in the right direction. We can supply these nudges using "hints".

You can define hints using the data-tile-type attribute, instead of defining a single type, you define multiple.

Some examples:

  • data-tile-type="s" this always shows a small tile.

  • data-tile-type="s,6m" this show a small tile, but shows a medium tile if the grid is more than 6 slots wide.

  • data-tile-type="m,5s,9_l" this shows a medium tile, from 5 slots wide it becomes a small tile, than starting from 9 slots it's turned into a large landscape tile.

The red grid below has 3 medium sized tiles. The grid is 5 slots wide (a small square tile occupies 1 slot), so in this situation we have an empty column on the right as tile number three does noet fit in the narrow column.

To resolve this, we can tell tilt to resize the last tile when the grid is 5 slots wide, the result is rendered as the green grid.

tilt set example

    <li><a class="tilt tilt-tile"><span>Foo</span></a>
    <li><a class="tilt tilt-tile"><span>Bar</span></a>
    <li><a class="tilt tilt-tile"><span>Baz</span></a>

</ul>
