Introduction

Fig is an animation library for animating photographs and captions. It unlocks a wide range of animations and effects for you to combine into amazing photo galleries.

Package Contents

  • A 'bin' folder containing the fig library itself.
  • A 'docs' folder containing detailed documentation and examples.
  • A 'src' folder containing the source files.

You'll probably only need the source files if you are an experienced webdeveloper yourself and want to modify the package. Note that there's no official support on modifying the package files, but, as you have just bought the package I find that you have right to access and modify the source code as you see fit.

Setup

Fig is easy to setup by adding both fig.css and fig.js (found in the 'bin' folder) to your webpage.

First copy the contents of the fig folder to your site folder.

Then copy the following HTML snippet to the <head> of your webpage.

<link href="fig.css" rel="stylesheet">

Then copy the following snippet and place it just before the closing </body> tag.

<script src="fig.js"></script>

Make sure the paths in the above code snippets match the location of the CSS and JS files.

The fig JavaScript runs automatically, so you don't have to worry about it. It finds fig elements on the page and configures them so they can be animated.

Quick Start

Now that you've successfully setup the files it's time to add your first fig.

<figure class="fig fig-frame">
    <figcaption class="fig-caption">
        <h1>Fig Pie</h1>
        <p>A yummie pie made out of delicious figs.</p>
        <p><a href="/docs">Read more</a></p>
    </figcaption>
    <img src="path/to/image/file.jpg" width="200">
</figure>

Not very exciting yet, but we can quickly change that. Alter the HTML as shown below.

<figure class="fig fig-frame" data-intro="fade" data-caption="fill:blur slide" data-interact="focus">
    <figcaption class="fig-caption">
        <h1>Fig Pie</h1>
        <p>A yummie pie made out of delicious figs.</p>
        <p><a href="/docs">Read more</a></p>
    </figcaption>
    <img src="path/to/image/file.jpg" width="200">
</figure>

We only added a CSS class and a couple of attributes and fig sets up the animations.

Browser Support

Fig works on all modern browsers.

It degrades gracefully on older browsers.

  • IE9 does not show animations.
  • IE8 renders a fallback where the caption is always above or below the image depending on the HTML setup.
